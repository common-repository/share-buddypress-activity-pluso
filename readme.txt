Restrict access to your website to users from certain countries.

== Description ==

<p>
<strong>This plugin is NOT compatible with most caching plugins.</strong>
</p>

<p>
Make sure you have NOT selected <strong>your own country</strong>. Otherwise, you will not be able to access your site.
</p>

<p>
This plugin helps you restrict access to your website to users from certain countries. You can choose counties which users can not get access to your website from.
</p>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Block by Country'
3. Activate the plugin from your Plugins page.
4. Select the countries you want to block.

= From WordPress.org =

1. Download 'Block by Country'.
2. Upload the 'Block by Country' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Block by Country' from your Plugins page.
4. Select the countries you want to block.
